استخدام نبراس في Chrome DevTools

الطريقة الأسهل والأكثر شيوعاً للحصول على تقارير فورية داخل المتصفح.

1

افتح Chrome DevTools

اضغط على F12 أو Ctrl+Shift+I

2

انتقل لعلامة التبويب

اختر "Lighthouse" من القائمة العلوية

3

حدد الفئات المطلوبة

اختر الأداء، إمكانية الوصول، SEO، والممارسات

4

انقر على تحليل

انتظر حتى تكتمل عملية التدقيق

استخدام نبراس من سطر الأوامر

مثالي للأتمتة والتكامل مع أنظمة CI/CD.

# التثبيت العالمي
npm install -g lighthouse

# تشغيل تحليل
lighthouse <url>

# خيارات متقدمة
lighthouse <url> --output=json --output-path=report.json

الاستخدام البرمجي

دمج نبراس في تطبيقاتك وأنظمة التطوير.

import lighthouse from 'lighthouse';
import { launch } from 'chrome-launcher';

const chrome = await launch();
const results = await lighthouse('https://site.com', { port: chrome.port });
